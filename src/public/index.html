How to connect

Use FileZilla, WinSCP or any FTP client with these settings:

  1. Set credentials & port, click Save & Apply
  2. Click Enable FTP
  3. In your FTP client: Host / Port / Username / Password from left
  4. Protocol: FTP (plain, not SFTP)

Each server has its own independent FTP port.

Setup Guide

  1. Create a bot at Discord Developer Portal
  2. Enable Message Content intent in Bot settings
  3. Copy the Bot Token and paste it above
  4. Invite the bot to your Discord server with Admin permissions
  5. Copy your Discord Server ID (Guild ID) and paste above
  6. Click Validate then Connect

The bot will auto-create roles and channels in your Discord server upon connection.
